YouTube видео в качестве фона на сайте

Сегодня научимся создавать фон на сайте из видео на YouTube с помощью jQuery плагина YTPlayer.

Смотреть урок на YouTube

Плагин jquery.mb.YTPlayer: pupunzi/jquery.mb.YTPlayer

Стартер для верстки start_html: На GitHub

Внимание! Для выполнения данного урока можно взять стартер "_clean_html" из архива "start_html" на GitHub и работать с ним, как было показано в видеоуроке.

Сайт плагина current-device: matthewhudson.github.io/current-device/. Скачать/сохранить исходник последней версии плагина можно по ссылке.

Обратите внимание, что все дополнительные плагины (jquery.mb.YTPlayer и current-device.min.js) вам необходимо подключить в проект самостоятельно, до вашего основного JS файла "common.js", если вы используете стартер "_clean_html".
