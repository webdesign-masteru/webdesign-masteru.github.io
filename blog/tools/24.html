Плавный скролл до блока, навигация по странице и кнопка Наверх

Научимся создавать плавную навигацию по странице (Landing Page), плавный скролл до нужного блока, реализуем кнопку "Наверх" с помощью jQuery плагина PageScroll2id. Данный плагин имеет множество настроек и умеет подсвечивать активные пункты меню.

Смотреть урок на YouTube

Архив start_html: На GitHub

Страница плагина PageScroll2id на GitHub: malihu/page-scroll-to-id

Последнюю минифицированную версию плагина можно взять из директории "github.com/malihu/page-scroll-to-id/tree/master/js/minified/" на GitHub.

Внимание! Для выполнения данного урока можно взять стартер "_clean_html" из архива "start_html" и работать с ним, как было показано в видеоуроке. Некоторые нужные плагины уже подключены в данном стартере.
